# 🌐 Online Exam System

## 📌 Project Description
The Online Exam System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to register, log in, attempt a quiz on Cloud Computing, and view their results instantly.

---

## 🚀 Features
- 🔐 User Registration and Login System
- 🏠 Dashboard Interface
- ☁️ Cloud Computing Quiz (10 Questions)
- ⏱ Timer-based Examination
- 📊 Instant Result Display
- 💾 Local Storage for user data

---

## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling & UI Design)
- JavaScript (Logic & Functionality)
- Git (Version Control)
- GitHub (Cloud Hosting)

## ▶️ How to Run the Project
1. Download or clone the repository
2. Open the project folder in VS Code
3. Run `login.html` using browser or Live Server
4. Register a new user
5. Login and start the quiz
